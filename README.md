# React Notes App

A Notes App built with React and Vite. Full CRUD functionality with localStorage persistence.

🔗 **Live Demo:** [React Notes App](https://react-notes-app-fatima.netlify.app/)

## Features

- Add notes
- Edit notes
- Delete notes
- Data persists across page refresh

## Tech Stack

- React
- Vite
- CSS
- localStorage

## How to Run

1. Clone the repo
2. `npm install`
3. `npm run dev`

## What I Learned

- Components and Props
- useState — dynamic data management
- useEffect — side effects and localStorage
- Race condition fix with isLoaded flag
- Full CRUD in React
